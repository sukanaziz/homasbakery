# Homas Bakery — Full Stack E-Commerce Platform

A fully deployed full-stack e-commerce web application built for a local bakery.
Live at: https://homasbakery.com

## Features
- Multi-item ordering system with pickup and delivery options
- Admin dashboard with protected routes and session-based authentication
- REST API backend for products, orders, and contact form submissions
- SQLite database for persistent data storage
- Input validation and error handling on both frontend and backend

## Tech Stack
- **Backend:** Node.js, Express, REST APIs
- **Database:** SQLite
- **Frontend:** JavaScript, HTML, CSS
- **Tools:** Git, GitHub, VS Code

## How to Run Locally
1. Clone the repo
2. Run `npm install`
3. Run `node server.js`
4. Open http://localhost:3000
